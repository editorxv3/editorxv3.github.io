# Ayaan Bukhari — Portfolio Website

This is your finished site: real HTML/CSS/JS, no build step, no frameworks.
Everything (videos, logos, code) is already inside this folder — you can host
it for free in a couple of minutes.

## Fastest option: Netlify (drag & drop, ~1 minute)

1. Go to https://app.netlify.com/drop
2. Drag this whole folder (`ayaan-bukhari-portfolio`) onto the page
3. Netlify gives you a live link immediately (like `random-name-123.netlify.app`)
4. Optional: in Site settings → Domain management, you can rename the
   subdomain for free, or connect a real `.com` domain if you buy one later

## Alternative: Vercel

1. Go to https://vercel.com/new
2. Choose "Deploy without Git" / drag-and-drop upload
3. Upload this folder — same result, a free live link

## Alternative: GitHub Pages

1. Create a new GitHub repository
2. Upload all these files to it (including the `assets` folder)
3. Go to Settings → Pages → set source to your main branch
4. Your site goes live at `yourusername.github.io/reponame`

## What's inside

- `index.html` — the page structure and content
- `styles.css` — all the design and layout
- `script.js` — the scroll-synced nav + video play buttons
- `assets/video/` — your showreel + 3 work clips (compressed for web)
- `assets/img/` — the 6 channel logos + Lazawal Ishq logo
- `assets/posters/` — thumbnail frames shown before each video plays

## Editing later

Everything is plain HTML/CSS — open `index.html` in a text editor (or paste
sections into a chat with Claude) to update text, swap videos, or add new
work samples. No build tools, no npm install, nothing to compile.
